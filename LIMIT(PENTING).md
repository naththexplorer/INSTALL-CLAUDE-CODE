# Rate Limit Usage Policy

## Batas Pemakaian Normal

* Penggunaan normal: **1–2 terminal**
* Maksimal: **700 prompt per 5 jam**

## Weekly Limit

* Batas mingguan: **6000 prompt / minggu**

## Jika Mencapai Limit

* Jika terkena **weekly limit**, silakan **hubungi admin**
* Jika limit terjadi:

  * **Lebih dari 3 hari** → akan diberikan **API pengganti**
  * **Kurang dari atau sama dengan 3 hari** → **harus menunggu hingga reset**

## Catatan

* Penggunaan berlebihan di luar batas wajar dapat dibatasi
* Gunakan API secara efisien sesuai kebutuhan
